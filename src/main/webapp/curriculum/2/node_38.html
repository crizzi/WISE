OBSERVING PROBABILITY

As helpful as the model is for understanding probability, it's not fun counting all those test offspring!

Instead, geneticists use a short cut: A tool invented in the early 1900s by Reginald C. Punnett, called a PUNNETT SQUARE.

How to read a Punnett Square

Punnett squares show all possible combinations of genotypes that can result from the parents' genotypes.

Use this Punnett square to test different combinations of parent genotypes. Observe the possible genotypes of the offspring.
